# Most-In-One Vuln Tester

A modular and easy-to-use web vulnerability testing tool designed to help security enthusiasts and professionals detect common web application vulnerabilities in a controlled environment.

---

## Features

- Detects common web vulnerabilities such as:  
  - SQL Injection (SQLi)  
  - Cross-Site Scripting (XSS)  
  - Directory Traversal  
  - Security misconfigurations  
- Modular architecture for easy extension  
- Web-based interface with authentication  
- Simple setup and usage  

---

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sudop3x/Most-In-One-Vuln-Tester.git
   cd Most-In-One-Vuln-Tester/Tool
   
2.Run the application:

    python app.py

3.Access the development server in your browser at:
 ```
 http://127.0.0.1:5000
```
4. Log in with the following credentials:
   
 ```


Username: peniel
Password: peniel

```
5.Usage

After login, use the web interface to scan authorized web applications for vulnerabilities.

The tool will test for several common vulnerability types and display results in an easy-to-understand format.

Requirements

Python 3.x

Recommended to use a virtual environment like VENV
